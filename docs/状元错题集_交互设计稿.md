# 状元错题集 · 交互设计稿

> **版本**：V1.0.0
> **更新日期**：2026/06/13
> **设计风格**：新中式文具 · 书卷气 · 成长的印记

---

## 一、设计理念与美学方向

### 1.1 Concept Direction — 新中式文具风

错题集自古是学子必备的「积累之器」。我们以「**新中式文具风**」为设计内核——将传统文房四宝的雅致与现代数字产品的简洁融为一体，传递「书卷气」与「成长的印记」。

**视觉隐喻**：宣纸的温润、墨迹的沉静、书页的留白、印章的庄重。

### 1.2 色彩体系

```css
:root {
  /* 主色 — 墨绿（守信、专注、成长） */
  --color-primary: #2D5A4A;
  --color-primary-light: #3D7A64;
  --color-primary-dark: #1D3A32;

  /* 点缀色 — 朱砂红（活力、成就、标记） */
  --color-accent: #C75450;
  --color-accent-light: #E86B67;
  --color-accent-pale: #FAF0EF;

  /* 中性色 — 宣纸色系 */
  --color-paper: #FAF7F2;
  --color-paper-dark: #F0EBE3;
  --color-ink: #2C2C2C;
  --color-ink-light: #6B6B6B;
  --color-ink-faint: #A8A8A8;

  /* 功能色 */
  --color-success: #3D8B6E;
  --color-warning: #D4864A;
  --color-error: #C75450;
  --color-info: #5A7FA8;
}

/* 深色模式 */
[data-theme="dark"] {
  --color-bg: #1A1E21;
  --color-surface: #252A2E;
  --color-paper: #2A2E32;
  --color-ink: #E8E4DE;
  --color-ink-light: #A8A8A8;
}
```

### 1.3 字体规范

| 用途 | 字体 | 字重 | 说明 |
|------|------|------|------|
| 品牌/标题 | Source Han Serif SC（思源宋体） | 600 | 传递书卷气，大标题使用 |
| UI 标题 | Noto Sans SC | 500 | 界面标题、次级标题 |
| 正文 | Noto Sans SC | 400 | 正文内容 |
| 数字/统计 | DIN Alternate / HarmonyOS Sans | 700 | 数字强调，卡片统计 |

### 1.4 间距系统

```
4px 基础单位
xs: 4px | sm: 8px | md: 16px | lg: 24px | xl: 32px | 2xl: 48px | 3xl: 64px
```

### 1.5 圆角与阴影

```
圆角 radius：
  - sm: 4px（标签、小按钮）
  - md: 8px（卡片、输入框）
  - lg: 12px（弹窗、大卡片）
  - full: 9999px（胶囊标签）

阴影：
  - subtle: 0 1px 3px rgba(45, 90, 74, 0.08)
  - medium: 0 4px 12px rgba(45, 90, 74, 0.12)
  - elevated: 0 8px 24px rgba(45, 90, 74, 0.16)
```

### 1.6 动效原则

| 场景 | 动效 | 参数 |
|------|------|------|
| 页面转场 | slide + fade | 300ms ease-out |
| 列表加载 | staggered fade-in | 每项 50ms delay，opacity 0→1，translateY 12px→0 |
| 按钮点击 | scale 0.96 → 1 | 150ms spring |
| 弹窗出现 | scale 0.92 → 1 + fade | 250ms spring |
| 底部导航切换 | crossfade | 200ms ease |
| 筛选标签展开 | height + fade | 250ms ease |
| Toast 提示 | slideUp + fade | 200ms ease |

## 二、页面结构与导航

### 2.1 底部导航栏（Tab Bar）

位于页面底部，高度 56px（不含安全区域），上边框 1px --color-paper-dark。

| 图标 | 文案 | 说明 |
|------|------|------|
| 🏠 首页 | 首页 | 概览统计 |
| 📝 错题 | 错题 | 列表筛选 |
| ➕ 添加 | 添加 | 拍照导入（主色调高亮） |
| 👤 我的 | 我的 | 设置同步 |

**视觉规范**：
- 当前选中项：图标填充色 --color-primary，文案 --color-primary，字重 500
- 未选中项：图标描边色 --color-ink-light，文案 --color-ink-light，字重 400
- 中间「添加」按钮：圆形突出，尺寸 52mm，背景 --color-primary，图标白色

│   │ 📱 手机号输入框            │   │

│   │        登录                 │   │
│   └────────────────────────────┘   │

└────────────────────────────────────┘
```

**视觉细节**：
- 背景：--color-paper 渐变至 --color-paper-dark（顶部深，向下渐浅）

**交互**：
- 手机号输入：11位数字校验，错误时输入框下方红色提示
- 获取验证码：60s 倒计时，防止重复点击
- 登录成功：scale 缩放 + 渐隐，300ms 后跳转首页

## 四、组件规范

### 4.1 按钮（Button）

| 类型 | 样式 | 适用场景 |
|------|------|---------|
| Primary | 填充 --color-primary，白字 | 主操作：登录、保存、导出 |
| Secondary | outline --color-primary，主色字 | 次操作：取消、重拍 |
| Ghost | 透明底，主色字 | 辅助操作：编辑、删除 |
| Danger | 填充 --color-error，白字 | 危险操作：退出、删除 |
| Disabled | 灰色背景，不可点击 | 不可用状态 |

**尺寸**：小 32px，中 40px，大 48px

### 4.2 输入框（Input）

```
状态：
- Default：底部边框 --color-ink-faint
- Focus：底部边框 --color-primary，label 上移
- Error：底部边框 --color-error，下方红色提示
- Disabled：背景 --color-paper-dark，文字灰色
```

### 4.3 标签（Tag）

| 类型 | 样式 |
|------|------|
| Default | 背景 --color-ink-faint 20%，文字 --color-ink-light |
| Primary | 背景 --color-primary 10%，文字 --color-primary |
| Success | 背景 --color-success 10%，文字 --color-success |
| Warning | 背景 --color-warning 10%，文字 --color-warning |
| Error | 背景 --color-error 10%，文字 --color-error |

### 4.4 卡片（Card）

- 背景：白色（深色模式 --color-surface）
- 圆角：radius lg（12px）
- 阴影：--subtle
- 内边距：md/lg（16-24px）

### 4.5 底部弹窗（Bottom Sheet）

- 从底部弹出，顶部圆角 radius lg
- 遮罩层：黑色 50% 透明度
- 拖拽指示条：顶部 40px 宽，4px 高，--color-ink-faint
- 进入：slide-up + fade，250ms spring
- 离开：slide-down + fade，200ms ease

### 4.6 Toast 提示

| 类型 | 样式 |
|------|------|
| Success | 绿色图标 + 文字提示，2s 后自动消失 |
| Error | 红色图标 + 文字提示，需要用户确认或手动关闭 |
| Info | 蓝色图标 + 文字提示，3s 后自动消失 |
| Loading | 旋转图标 + 文字提示，不自动消失 |

### 4.7 空状态（Empty State）

- 居中布局，插画 120x120 + 标题 + 描述
- 插画风格：简约线性插画，与品牌调性一致
- 按钮：Primary 按钮引导操作

## 七、打印排版规范

```
