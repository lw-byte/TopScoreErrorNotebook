## 1. 项目概述

### 1.1 项目背景

状元错题集是一款面向中国 K12 学生的**智能错题整理与学习管理工具**，解决传统纸质错题本整理效率低、难以检索、无法跨设备同步、打印不便等痛点。

### 1.2 项目目标

| 指标 | 目标值 | 说明 |
|------|--------|------|
| 拍照到识别完成 | < 5s | 4G 网络环境 |
| 页面首次加载 | < 2s | 4G 网络环境 |
| OCR 印刷体准确率 | > 95% | 阿里云 OCR |
| OCR 手写体准确率 | > 85% | 阿里云 OCR |
| 离线支持 | ≥ 7 天 | 断网情况下本地数据可用 |

### 1.3 上线时间窗口

- **开发周期**：约 12 周
- **目标上线**：2026/Q3

## 6. 风险与依赖

### 6.1 技术风险

| 风险 | 影响 | 发生概率 | 应对措施 |
|------|------|---------|---------|
| 阿里云 OCR 识别准确率不足 | 高 | 中 | 提供手动编辑入口；置信度低时提示用户 |
| 弱网环境下同步失败 | 中 | 中 | 本地优先+重试队列；显示同步状态 |
| 大图片导致 OOM | 中 | 低 | 压缩至 1080p；使用 Coil 缓存 |
| 多设备同步冲突 | 中 | 低 | 弹出冲突处理 UI；默认保留本地最新 |
| 导出格式兼容问题 | 中 | 低 | 测试主流 Office 版本；提供备用格式 |

### 6.2 外部依赖

| 依赖项 | 类型 | 风险 |
|--------|------|------|
| 阿里云 OCR 服务 | 第三方 API | 服务不可用时降级提示 |
| 阿里云 OSS 服务 | 第三方存储 | 上传失败时本地缓存重试 |
| 后端 API | 自建服务 | 需提前约定接口规范 |

### 6.3 待确认事项（技术设计文档第 12 章）

| 编号 | 问题 | 影响 | 状态 |
|------|------|------|------|
| Q1 | 阿里云 OCR 具体产品选型 | OCR 识别效果 | 待确认 |
| Q2 | OCR 是否区分印刷体和手写体 | OCR 识别效果 | 待确认 |
| Q3 | 导出库具体选型 | 导出功能 | 待确认 |
| Q5 | 多设备同步冲突默认策略 | 数据一致性 | 待确认 |
| Q6 | 是否需要微信登录 | 登录方式 | 待确认 |
